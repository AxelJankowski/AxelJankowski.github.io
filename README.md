# AxelJankowski.github.io

Personal portfolio website built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run build
npm run preview
```

## Deployment

The site deploys automatically to GitHub Pages on push to `master` via GitHub Actions.

**One-time setup:** In the repository settings, set **Pages → Build and deployment → Source** to **GitHub Actions**.

## Archive

The original 2019 static site is preserved on the `legacy/2019` branch and the `v1-legacy` tag.
